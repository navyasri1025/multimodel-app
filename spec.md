# Project Specification

## Project Name
MultiModel App

## Objective
Build a Streamlit application that allows users to compare responses from multiple AI models.

## Features
- Compare multiple models
- User-friendly interface
- Secure API key management using .env
- Response latency display

## Technologies
- Python
- Streamlit
- OpenRouter API

## Future Enhancements
- Chat history
- Dark mode
- Export responses
